无聊加条线惹的祸……@无聊小球

这几天经常没事就做做小球运动，今天学着书做了个小球缓动过程，也就是:

(1)设定一个比例系数（<1）；

(2)设定目标点；

(3)计算物体到目标点之间的距离；

(4)将距离与比例系数相乘，得到速度；

(5)加入速度到当前物体的位置；

(6)重复3-5步；
那么这小段代码便是：

private var easing:Number=0.2;
private var targetx:Number=stage.stageWidth/2;
private var targety:Number=stage.stageHeight/2;

function onEnterFrame(event:Event):void{
var dx:Number=targetx-ball.x;
var dy:Number=targety-ball.x;
vx=(targetx-ball.x)easing;
vy=(targety-ball.y)easing;
ball.x+=vx;
ball.y+=vy;
这段代码虽然简单（看上去）那是因为我还没加上其他东西，但是这样是运行不了的。 然后边就是我前面那篇的纠结之处了，我开始做了一个Ball类文件Ball.as,那是一个添加一个球（圆形）的方法，为以后做无聊球运动是方便引用的。 然后为这个小球缓动运动fla（空白的）设置了一个文档类Scale1.as。但是Ball怎么设置，我像昨天那样设置类路径，但是根本搞不清其中的关系，该import什么，文件该怎么放，包怎么定义，快崩溃了，不过还是静下心来仔细思考，并且看书-“面向对象编程”那章。还是用了很多时间翻书、尝试才把Ball类文件用到scale1.as上，原来文档类所在文件夹就是包名，而且类文件要和as放在同一个根目录上；假如fla和as文件位于同一个目录内，那么就是在同一个包内，就不用命名。然后把ball放在了fla所在（根）目录上的一个oneball文件夹上，然后在fla文件actionscript3.0定义设置源路径上要加入oneball这个目录位置（flash cs3上到文件-发布设置-actionscript版本后面的“设置”，flashcs5到fla文件点选右边属性-脚本actionscript3.0后面的一个扳手图标），设置完后然后在Scale1.as上的顶端、定义包下面加入import oneball Ball;就可以用Ball方法添加小球了。

Ball.as代码：

package{
    import flash.display.Sprite;

public class Ball extends Sprite{
    public var radius:Number;
    private var color:uint;

    public function Ball(radius:Number=40,color:uint=0x116666){
        this.radius=radius;
        this.color=color;
        init();
    }
    public function init():void{
        graphics.beginFill(color);
        graphics.drawCircle(0,0,radius);
        graphics.endFill();
    }
}

}
然后下面是Scale1.as代码：

package {
    import oneball.Ball;
    import flash.events.Event;
    import flash.display.Sprite;

public class Scale1 extends Sprite{
    private var ball:Ball;
    private var scale:Number=0.2;
    private var targetx:Number=stage.stageWidth/2;
    private var targety:Number=stage.stageHeight/2;
    private var vx:Number;
    private var vy:Number;

    //添加小球
    public function Scale1() {
        init();
    }

    private function init():void{
        ball=new Ball();
        addChild(ball);
        addEventListener(Event.ENTER_FRAME,onEnterFrame);
    }
    function onEnterFrame(event:Event):void{
        var dx:Number=targetx-ball.x;
        var dy:Number=targety-ball.x;
        vx=(targetx-ball.x)*scale;
        vy=(targety-ball.y)*scale;
        ball.x+=vx;
        ball.y+=vy;
    }
}

}
然后问题又来了，那就是缓动计算那段代码是一直计算的，因为这就好比每次都需一半，取无数次都取不完的。所以要添加一段停止代码。当小球距离目标点距离小于1像素时停止缓动，直接到达目标点。

最后最后，就是无聊的后果了，又加了个鼠标拖动小球代码，然后不过瘾想加条直线连着小球与目标点，那就不断地“画”直线吧，从目标点到小球点，且在画下一条直线时要把之前的那条清除掉。

恩，测试播放效果不错，恩，鼠标拖动时没线感觉假假的，所以又在mousemove函数上加了个不断画线清楚的addline代码，但问题来了，enterFrame函数不知怎么去除，在每次拖动完一次后就不知怎么会一直画线（鼠标位置到目标点）。在mouseup上去除，他就一开始就检测，因为那时还没定义addline，所以错误了。悲剧了，试了很多个地方都不行，悲剧了。

最后还是突然醒悟，既然它一直画直线从鼠标位置到中心点，那么就把鼠标位置该为小球位置就ok啦！

改完之后终于大功告成了，松了口气……

下面是改完后的Scale2.as完整代码：

package {
    import oneball.Ball;
    import flash.events.Event;
    import flash.display.Sprite;
    import flash.events.MouseEvent;
    import flash.ui.Mouse;

public class Scale2 extends Sprite{
    private var ball:Ball;
    private var scale:Number=0.2;
    private var targetx:Number=stage.stageWidth/2;
    private var targety:Number=stage.stageHeight/2;
    private var vx:Number;
    private var vy:Number;

    //添加小球
    public function Scale2() {
        init();
    }

    private function init():void{
        ball=new Ball();
        addChild(ball);
        ball.addEventListener(MouseEvent.MOUSE_DOWN,onMouseDown);
        addEventListener(Event.ENTER_FRAME,onEnterFrame);
    }

    //鼠标拖动
    //鼠标按下
    private function onMouseDown(event:MouseEvent):void{
        ball.startDrag();
        stage.addEventListener(MouseEvent.MOUSE_UP,onMouseUp);
        stage.addEventListener(MouseEvent.MOUSE_MOVE,onMouseMove);
        removeEventListener(Event.ENTER_FRAME,onEnterFrame);
    }

    //鼠标移动
    private function onMouseMove(event:MouseEvent):void{
        addEventListener(Event.ENTER_FRAME,addline);
        function addline(e:Event):void{
            graphics.clear();
            graphics.moveTo(targetx,targety);
            graphics.lineStyle(1);
            graphics.lineTo(ball.x,ball.y);
        }
    }

    //松开鼠标
    private function onMouseUp(event:MouseEvent):void{
        ball.stopDrag();
        addEventListener(Event.ENTER_FRAME,onEnterFrame);
        stage.removeEventListener(MouseEvent.MOUSE_UP,onMouseUp);
        stage.removeEventListener(MouseEvent.MOUSE_MOVE,onMouseMove);
    }

    //重复计算速度--缓动(增加停止条件）
    function onEnterFrame(event:Event):void{
        var dx:Number=targetx-ball.x;
        var dy:Number=targety-ball.x;

        //停止缓动
        if(Math.abs(dx)<1||Math.abs(dy)<1){
            ball.x=targetx;
            ball.y=targety;
            removeEventListener(Event.ENTER_FRAME,onEnterFrame);
            trace("停止“缓动计算”成功")
        }else{
            graphics.clear();
            vx=(targetx-ball.x)*easing;
            vy=(targety-ball.y)*easing;
            ball.x+=vx;
            ball.y+=vy;

            //无聊加条线
            graphics.moveTo(targetx,targety);
            graphics.lineStyle(1);
            graphics.lineTo(ball.x,ball.y);
        }
    }
}

}
