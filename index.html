Sketch 插件清单

1. Looper

快速旋转、复制图形。
插件官网

2. Craft by InVision

必备设计师工具。
Craft是一套提升效率的组合插件，支持 Sketch 和 PS。可以 自动填充图片／文本数据、可以快速复制排列元素、可以制作组件库使用。

插件官网

3. Confetti

一键生成随机图案背景。
插件官网

4. Magic Mirror 2

实现类似PS里的「自由变换」功能。
插件官网

5. Map Generator

将一个位置的真实地图填充到图层 (来源Google地图)。
通过插件可以快速制作地图展示样式在一个App或者网站里面。你只需要提供一个位置以及地图展示CSS样式。
插件官网

6. Compo

Compo 实现了 Sketch 中的动态按钮、动态组件。通过 Compo ，按下 ⌘+J 快捷键将文本变为按钮或者存在的组件。可以配合 State Machine 插件使用。
插件官网

7. Slinky

导出 Sketch 设计为 HTML 样式，可用于邮件模版。
插件官网

8. Fluid

Fluid 支持在 Sketch 中的元素定义更多的约束，用于响应式设计。 此基于 Auto Layout 的约束功能。
插件官网

9. Auto Layout

在 Sketch 中完成响应式设计。
A Sketch Plugin that integrates seamlessly into Sketch and enables defining and viewing different iPhone/iPad sizes including Portrait/Landscape.
插件官网

10. Launchpad

Launchpad 支持通过 Sketch 发布响应式网站。 无需代码。
插件官网

11. CSSketch

CSSketch is a Sketch 3 plugin that enables you to modify your designs quickly by attaching it a stylesheet to it that you can use to lay out your designs, change colors and shadows quickly, etc. It supports Sass and less stylesheets, which means you can put functions and variables in your stylesheet to make your workflow even more efficient.
插件官网

12. Social Artboards

将不同社交平台的图片尺寸作为 Artboard 画板尺寸预设添加到 Sketch。
插件官网

13. Day Player

支持从一些图片提供网站添加图片到Sketch里。
插件官网

14. Find and replace

查找并替换文本，全文档匹配。
插件官网

15. FlipSize

小工具，用于互换画板、图层、形状的宽高。
插件官网

16. Pixel Perfector

用于对齐像素，相比Sketch里的默认整数，这个插件还支持0.5、2这样的倍数。
插件官网

17. Random sizing

A Sketch 3 plugin that will randomly size the width or height of selected elements. In combination with other plugins like Duplicator, this is useful to simulate graphs or charts of random values.
插件官网

18. Remove All Disabled Styles

A simple plugin that removes all disabled styles from all layers in the current selection. Useful to clean up your document from stray styles. Does not remove enabled styles (only the styles with a [ ] )
插件官网

19. Rename It

支持重命名多个图层。快捷键⌃+⌘+R。
插件官网

20. Repeat Objects

支持自动复制元素到所有画板。
插件官网

21. Lorem-Ipsum

This is a plugin for quickly creating Lorem Ipsum text in Sketch.
插件官网

22. Sketch Icon Stamper

一键生成 iOS ICON 的所有尺寸。
插件官网

23. Runner

提供 Sketch 的快捷搜索命令操作，类似Spotlight。
插件官网

24. Select Child Layers

快速选择图层组里面的所有子元素。（常规操作是打开图层组，通过 Shift 键全选）
插件官网

25. Sketchy Pies

制作饼图的一个插件。
插件官网

26. Sort Me

Sort Me 支持根据图层名称自动排序图层。
插件官网

27. Content Generator

自动填充图片内容，同功能已集成到 Craft。
插件官网

28. Sketch Measure

除了标注功能外，还支持导出标注规范文件。
插件官网

29. Sketch Search Everywhere

支持搜索并定位图层。快捷键Alt + Command + F。
插件官网

30. Sync to Slides

支持同步 Sketch 画板到 Google Slides。
插件官网

31. Appalette

支持导入色板，从各种（国外）App。
插件官网

32. Sketch Move Half Pixel

支持半像素移动图层。
插件官网

33. Skatter

用于自动创建填充图案。
插件官网

34. IconFlower

以叶序“花状”自动排列 ICON。
插件官网

35. Sketch Plugin Manager

自动管理插件，通过Git，实现插件自动更新。
插件官网

36. Bootstrap grid

支持插入正确的删格化参考线网格。
插件官网

37. Cleanup Useless Groups

用于整理图层，清除空图层组、合并太深的无用嵌套图层。
插件官网

38. Nudge Corner Radius

支持通过快捷键快速改变圆角。
插件官网

39. Spell Check Whole Page

拼写检查，支持全页面图层检查。
插件官网

40. Sketch Import Symbols

支持从另外一个 Sketch 文件导入 Symbols。
插件官网

设计每周清单：Sketch 45 版本更新、Facebook Sound Kit以及如何设计声效、Google News 全新设计

设计每周清单专注于互联网设计领域内容，分为热点资讯、软件工具、设计经验、案例研究、网站推荐、深度思考等栏目。本文同时发布于知乎专栏「Designer News」，欢迎关注。

热点资讯

  1. Sketch 45 版本更新：演示模式改为了隐藏UI功能，支持内部更新插件和禁用不兼容的插件、绘制矢量图形时锚点支持智能辅助线、新的颜色选择器方便管理大量颜色。/ 译文
  2. Touch Bar Preview：如果你尝试为 Touch Bar 做一些设计？也许这个 Touch Bar 预览工具可以帮到你，调试设计稿。

软件工具

  1. Gravit Designer：软件还特地适配了中文，支持导入Sketch文件，Sketch拥有的功能它基本都有，操作界面还基本一样。但缺少 Sketch 目前众多的插件支持（也是 Sketch 的优势所在）。Gravit Designer的优势就是跨平台（Windows、Mac、Linux、Chrome OS），而且软件本身免费。/ 软件功能介绍
  2. Facebook Sound Kit ：Facebook Design 网站推出的 Sound Kit 资源，网页下方可以在线预览各个音效。下载的资源包分为高音量（电脑上使用）和低音量（手机中使用）两种。至于版权，只可以用于非商业目的的使用。

设计经验

  1. Facebook 如何提高移动设计中的声效设计：由 Facebook 声效设计师 Will Littlejoh（没错他们竟然有这种设计师）写的文章，介绍道很多人会忽略声音在产品设计中起到的重要角色。声效设计师其实是给用户带来的是界面与真实世界的承接，你的现实感知很大程度上依赖于所听到的声音。
  2. 任天堂 Switch 游戏主机的 UI 思考：发售就大卖的任天堂新一代主机/掌机 Switch，玩的时候总会感慨其UI以及设计真的做得很棒。
  3. 常用心理学设计定律：Von Restorff effect 雷斯托夫效应、Serial position effect 序列位置效应、Cognitive load 认知负荷、Hick’s Law 席克法则、Law of Proximity 接近法则，文中举例介绍了这几条设计定律。/ 另外推荐雅虎UED交互原理漫画
  4. Framer 系列入门视频教程：Youtube 上 Sketch Together 推出的 Framer入门基础教程。Framer 在国外的人气渐高，新版由全代码改为可视化+半代码后，入门门槛降低了，也适合对Code+Design感兴趣的设计师。
  5. Framer 使用技巧：Framer官方博客文章，用动图介绍了一些 Framer 小技巧、快捷键的使用。
  6. Google News 全新设计：Google 对自己产品进行的全新改版升级，新的UI 更加清爽，更加专注于新闻内容。
  7. 色彩理论基础：介绍了8条色彩理论，适合了解掌握。/ 另一篇设计师色彩理论文章
  8. 麦当劳的微交互：从系列感的玩具，到活动感的玩具，到具有交互感的玩具，每个玩具都带给很多小朋友不同的期待与欢乐。

网站推荐

  1. WebVR Showroom：一个 3D+VR 的家装在线 VR 体验网站，浏览器上体验十分流畅，很真实，点击家具可以替换材质。

设计清单-2017.6

6月设计清单，Apple WWDC 2017 的举行，Dribbble正式推出 iOS 客户端，Sketch 45版本小更新，基本上这个月没有太多的大事件发生。但不妨碍好的设计文章产生，这个月较流行的设计经验文章依旧围绕于新软件、新技术、VR等趋势为主题。

另外一些工具清单如Sketch插件清单、设计规范清单，值得推荐收藏。
上个月Framer新版发布后，在国外的人气渐涨，新版由全代码改为可视化+半代码后，入门门槛降低了。适合对 Code+Design 感兴趣的设计师，可以看一下新版 Framer 的一些教程。

详细的可以查阅每周的设计清单。

矢量ICON设计快速技巧

作者介绍了一些AI绘制ICON的小技巧，有些还是和常规的思路不一样，值得参考。

在不同App、网站，即使是相同的ICON图形，有时也会因为细微区别而需要重新调整形状和风格，需要重新绘制。

所以是否存在绘制ICON的最优方法？以节省绘制时间。同时这也是一个不错的挑战，十分有趣。目前已经找到了一些常见ICON的绘制技巧，使用的是AI软件（以及作者分享的Greyprint模板），这些技巧也适用于其他软件，因为思路都是相通的。

爱心

创建一个爱心最快的方法其实只需要使用三个锚点组成一个v字形状，然后加上超大的描边。这种方法还有个优势，就是可以通过调整贝塞尔曲线控制柄，做成有弧度的爱心，这可以形成多种变化， 做出你想要的爱心形状。（有个缺点是如果不是尖角的心形，需要“轮廓化描边”等处理？）

WIFI

WIFI信号ICON可以通过绘制几个圆形，再布尔运算二而得。AI的路径查找器->分割。

链接

超链接ICON也类似，但是重叠修剪处需要有一定的规划才能处理好。

镭射

有时候需要跳出常规框架。例如虚线是十分有用的，特别是一个圆里面需要分割成多等分的时候。

雪花

AI的外观面板可以添加变换效果，用来复制物体（扭曲和变换->变换）。可以用来创建类万花筒效果，以及快速创建雪花图案，还可以实时修改。

盾牌
