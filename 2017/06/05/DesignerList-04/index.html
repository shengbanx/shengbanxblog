设计每周清单：天猫品牌全新升级、Framer发布新版本、Skype全新改版

设计每周清单专注于互联网设计领域内容，分为热点资讯、软件工具、设计经验、案例研究、网站推荐、深度思考等栏目。本文同时发布于知乎专栏「Designer News」，欢迎关注。

热点资讯

  1. 天猫品牌全新升级：在上一周5月23日，天猫 slogan 正式对外更名「理想生活上天猫」，相对之前「上天猫就够了」变得更加聚焦。同时，对品牌进行了全新升级。
  2. Skype全新改版：新功能，新外观，和目前大部分App改版一样，极简外观似乎成为了趋势。
  3. Framer发布新版本：终于在31日，Framer发布了新版本。作为一个原型软件，这一次将设计、原型、代码完美结合，让懂前端代码的设计师更充分发挥设计能力。 / DN讨论
  4. Apple 官方设计资源更新了 Adobe XD 文件：Apple UI Design Resources是苹果今年推出的官方设计资源，包含Sketch、Photoshop文件，目前更新了Adobe XD文件。
  5. Google 新 Emoji 表情 ：Google 在2017年 I/O 中发布的全新 emoji。/知乎讨论

软件工具

  1. Wake：一个分享设计工作与收集反馈的团队协作工具，开始为小团队提供免费版服务。
  2. Feather：一套简洁的开源线性ICON。

设计经验

  1. 各类移动端导航菜单的优缺点：讨论了时下几种移动端导航模式的优缺点：Hamburger Menu 侧栏导航、Tab Bar 底部导航、Priority+ Pattern 优先级动态菜单、Floating Action Button 浮动按钮、Full-Screen Navigation 全屏导航菜单、Gesture-Based Navigation 手势导航、3D Touch、Innovative Navigation Patterns 创新导航模式。
  2. 3种方式从Sketch导入到Framer：Framer发布新版本受到了广泛关注，很多人都表示已经以Sketch作为主力设计工具，那么如何与Framer共同使用呢？官方没有给出合适的方案（大概是要立志取代Sketch作为主力设计工具吧…）。这篇文章介绍了3种目前的方法，从Sketch导入到Framer，以及其优缺点对比。
  3. 麦当劳通用营养标识ICON：麦当劳为109个国家设计的通用营养标识ICON。
  4. 如何用AE制作粒子页面特效：使用Trapcode Particular粒子插件，在AE上制作粒子特效。

网站推荐

  1. Apple Accessibility：苹果的无障碍设计。

深度思考

  1. 设计师 Bill Kenney 专访：设计师出身的Bill，创立了Focus Lab公司，管理团队。
