矢量ICON设计快速技巧

作者介绍了一些AI绘制ICON的小技巧，有些还是和常规的思路不一样，值得参考。

在不同App、网站，即使是相同的ICON图形，有时也会因为细微区别而需要重新调整形状和风格，需要重新绘制。

所以是否存在绘制ICON的最优方法？以节省绘制时间。同时这也是一个不错的挑战，十分有趣。目前已经找到了一些常见ICON的绘制技巧，使用的是AI软件（以及作者分享的Greyprint模板），这些技巧也适用于其他软件，因为思路都是相通的。

爱心

创建一个爱心最快的方法其实只需要使用三个锚点组成一个v字形状，然后加上超大的描边。这种方法还有个优势，就是可以通过调整贝塞尔曲线控制柄，做成有弧度的爱心，这可以形成多种变化， 做出你想要的爱心形状。（有个缺点是如果不是尖角的心形，需要“轮廓化描边”等处理？）

WIFI

WIFI信号ICON可以通过绘制几个圆形，再布尔运算二而得。AI的路径查找器->分割。

链接

超链接ICON也类似，但是重叠修剪处需要有一定的规划才能处理好。

镭射

有时候需要跳出常规框架。例如虚线是十分有用的，特别是一个圆里面需要分割成多等分的时候。

雪花

AI的外观面板可以添加变换效果，用来复制物体（扭曲和变换->变换）。可以用来创建类万花筒效果，以及快速创建雪花图案，还可以实时修改。

盾牌

其实有时候并不存在快速的方法， 需要使用钢笔工具，手绘一个形状，例如这是我绘制的一个盾牌：

物体（Object）->复合路径（ Compound Path）->释放（Release）用来转换外描边成两个独立的路径（大小盾牌）。
