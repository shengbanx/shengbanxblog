设计每周清单专注于互联网设计领域内容，分为热点资讯、软件工具、设计经验、案例研究、网站推荐、深度思考等栏目。本文同时发布于知乎专栏「Designer News」，欢迎关注。

热点资讯

  1. 2017谷歌I/O大会提出AI First概念：在这次Google I/O 2017正式宣布谷歌战略从 Mobile First（移动先行）转向 AI First （人工智能先行）。对于互联网设计行业是否会带有一定的影响？例如 Google Assistant 中类似Siri一样的对话式 UI 交互。
  2. Sketch 44 版本更新：改进了Artboard的选择切换，并且支持Artboard内容大小自适应调整； 带来了自适应布局功能，可以减少对AutoLayout等自动布局插件的依赖；支持管理丢失的字体，替换为所选字体；改善了圆角处理方式，过窄过宽的角也能完美适应。
  3. Google公开了Auto Draw的所有数据样本 超过1500万用户在 之前Quick, Draw! 游戏中贡献了这5000万数据样本，用于训练 Auto Draw（根据绘制的图形，自动修正提供绘制好的图形的工具）。

软件工具

  1. SVGito：专门针对于Sketch导出的SVG文件，可以压缩文件大小的同时保持设计样式不改变。
  2. 交互原型工具清单：列出目前60+交互原型工具，大多都是可以快速制作原型、动效、导出到开发等。但推荐影响力比较大的几个：Adobe XD、Principle、ProtoPie、Axure、Flinto、Framer、InVision、UXPin、Kite Compositor、墨刀、Origami Studio、Craft、Launchpad、Webflow。感兴趣的可以阅读翻译整理的译文。
  3. Craft插件：Sketch插件，更新后支持直接在 Sketch 上制作原型。并且可以同步设计稿到InVision用于线上展示。
  4. Brand.ai：设计组件管理系统，可以轻松从库中拖动组件于Sketch中设计，同时支持导出到开发并自动同步样式。配合 UX Power Tools 使用效果更佳。

设计经验

  1. 移动端横向列表最佳实践：一些关于横向List需要注意的点：前后遮显关系的视觉暗示，使用“查看所有”按钮，响应式设计适应pad端浏览，在pad端只显示有限内容，滑动时会自动吸附到端点。
  2. AI矢量icon绘制技巧：介绍了一些不错的画icon小技巧。例如画爱心，一般人会想到勾外轮廓，实际上只需要三个锚点就可以了。而且这种方法还有个优势，可以直接调成贝塞尔曲线，即可改变爱心的弧度。另外可以查看作者收录的更多绘制技巧gif。
  3. 设计文件（Sketch）命名规范：第二部分细致说明了Sketch文件的命名以及它的图层的整理。
  4. 又一篇VR设计师指南：系统的列出了设计方法、设计工具、交互基础。
  5. Sketch动态ICON：设计一套ICON，和颜色模版，可以动态地修改ICON并使用不同的颜色。

案例研究

  1. Spotify 交互分析和Redesign：对Spotify应用进行分析，然后进行了自发性的Redesign，是个不错的案例研究。

设计灵感

  1. 聊天式UI：illusion.ai——提供对话式UI交互的工具。对比于常规网站的静态的内容呈现，这里采用了对话式气泡UI进行交互，动态展示用户需要的内容，有点类似游戏类h5的选项对话。让冰冷的网站更加亲切拟人化，应该可以适用于某些场景。
  2. 滑动多选交互：很多App都支持轻扫删除操作，是否可以有一个滑动多选的操作呢？链接中展示了在list上侧滑的同时上下滑动，选择更多内容一起执行操作的交互。

网站推荐

  1. Your Majesty：对一些Redesign后的公司品牌作介绍，如：Samsung SmartThings 、adidas 、BMW 等。
  2. UIPixels：免费Photoshop&Sketch源文件下载：Mockup / UI Kit / ICON
