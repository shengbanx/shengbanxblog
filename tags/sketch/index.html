Sketch 插件清单

1. Looper

快速旋转、复制图形。
插件官网

2. Craft by InVision

必备设计师工具。
Craft是一套提升效率的组合插件，支持 Sketch 和 PS。可以 自动填充图片／文本数据、可以快速复制排列元素、可以制作组件库使用。

插件官网

3. Confetti

一键生成随机图案背景。
插件官网

4. Magic Mirror 2

实现类似PS里的「自由变换」功能。
插件官网

5. Map Generator

将一个位置的真实地图填充到图层 (来源Google地图)。
通过插件可以快速制作地图展示样式在一个App或者网站里面。你只需要提供一个位置以及地图展示CSS样式。
插件官网

6. Compo

Compo 实现了 Sketch 中的动态按钮、动态组件。通过 Compo ，按下 ⌘+J 快捷键将文本变为按钮或者存在的组件。可以配合 State Machine 插件使用。
插件官网

7. Slinky

导出 Sketch 设计为 HTML 样式，可用于邮件模版。
插件官网

8. Fluid

Fluid 支持在 Sketch 中的元素定义更多的约束，用于响应式设计。 此基于 Auto Layout 的约束功能。
插件官网

9. Auto Layout

在 Sketch 中完成响应式设计。
A Sketch Plugin that integrates seamlessly into Sketch and enables defining and viewing different iPhone/iPad sizes including Portrait/Landscape.
插件官网

10. Launchpad

Launchpad 支持通过 Sketch 发布响应式网站。 无需代码。
插件官网

11. CSSketch

CSSketch is a Sketch 3 plugin that enables you to modify your designs quickly by attaching it a stylesheet to it that you can use to lay out your designs, change colors and shadows quickly, etc. It supports Sass and less stylesheets, which means you can put functions and variables in your stylesheet to make your workflow even more efficient.
插件官网

12. Social Artboards

将不同社交平台的图片尺寸作为 Artboard 画板尺寸预设添加到 Sketch。
插件官网

13. Day Player

支持从一些图片提供网站添加图片到Sketch里。
插件官网

14. Find and replace

查找并替换文本，全文档匹配。
插件官网

15. FlipSize

小工具，用于互换画板、图层、形状的宽高。
插件官网

16. Pixel Perfector

用于对齐像素，相比Sketch里的默认整数，这个插件还支持0.5、2这样的倍数。
插件官网

17. Random sizing

A Sketch 3 plugin that will randomly size the width or height of selected elements. In combination with other plugins like Duplicator, this is useful to simulate graphs or charts of random values.
插件官网

18. Remove All Disabled Styles

A simple plugin that removes all disabled styles from all layers in the current selection. Useful to clean up your document from stray styles. Does not remove enabled styles (only the styles with a [ ] )
插件官网

19. Rename It

支持重命名多个图层。快捷键⌃+⌘+R。
插件官网

20. Repeat Objects

支持自动复制元素到所有画板。
插件官网

21. Lorem-Ipsum

This is a plugin for quickly creating Lorem Ipsum text in Sketch.
插件官网

22. Sketch Icon Stamper

一键生成 iOS ICON 的所有尺寸。
插件官网

23. Runner

提供 Sketch 的快捷搜索命令操作，类似Spotlight。
插件官网

24. Select Child Layers

快速选择图层组里面的所有子元素。（常规操作是打开图层组，通过 Shift 键全选）
插件官网

25. Sketchy Pies

制作饼图的一个插件。
插件官网

26. Sort Me

Sort Me 支持根据图层名称自动排序图层。
插件官网

27. Content Generator

自动填充图片内容，同功能已集成到 Craft。
插件官网

28. Sketch Measure

除了标注功能外，还支持导出标注规范文件。
插件官网

29. Sketch Search Everywhere

支持搜索并定位图层。快捷键Alt + Command + F。
插件官网

30. Sync to Slides

支持同步 Sketch 画板到 Google Slides。
插件官网

31. Appalette

支持导入色板，从各种（国外）App。
插件官网

32. Sketch Move Half Pixel

支持半像素移动图层。
插件官网

33. Skatter

用于自动创建填充图案。
插件官网

34. IconFlower

以叶序“花状”自动排列 ICON。
插件官网

35. Sketch Plugin Manager

自动管理插件，通过Git，实现插件自动更新。
插件官网

36. Bootstrap grid

支持插入正确的删格化参考线网格。
插件官网

37. Cleanup Useless Groups

用于整理图层，清除空图层组、合并太深的无用嵌套图层。
插件官网

38. Nudge Corner Radius

支持通过快捷键快速改变圆角。
插件官网

39. Spell Check Whole Page

拼写检查，支持全页面图层检查。
插件官网

40. Sketch Import Symbols

支持从另外一个 Sketch 文件导入 Symbols。
插件官网
